## Paying Attention to detail
when using CSS and Html in general Even if you write the property the right way. For example ```<h1> title </h``` with the closing bracket missing the code won't work. with CSS properties you first state the property. like ``` align-self: center;``` if I put the colin : next to the property. But then forget the ; next to the element I want to set the property too. The property won't take action and won't work.  This can be really frustrating when you know what property you are appealing and how to do it but forget. A small detail so you have to pay attention to that detail.
